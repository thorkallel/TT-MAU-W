Desarrolado en

Reactjs
React Hooks
React Context
React Reducer

Para instalar la aplicación se debe ejecutar

npm install

Para correr la aplicación se debe ejecutar

npm start

Se puede visualizar la aplicacion ingresando al directorio /build y ejecutando live-server en el puerto :8080

http://127.0.0.1:8080/
